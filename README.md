# vardel
{vardel}, or Variance Decomposition Light, is a smaller version of the experimental {varde} and {agreement} packages originally authored by [Dr. Jeffery Girard](https://github.com/jmgirard), the University of Kansas. 

This package is instrumental for running parallelized simulations of the experimental Generalized Intraclass Correlation measure (Girard & Simmons, 2026).
